% Abstract
\begin{abstract}
This paper presents the development of a Retrieval-Augmented Generation (RAG)-based chatbot system to enhance thesis literature search in the CSPC library. The system integrates a Large Language Model (LLM) with a vector database to provide efficient, topic-oriented search capabilities. This document outlines the methodology, results, and discussion of the system's implementation.
\end{abstract}
